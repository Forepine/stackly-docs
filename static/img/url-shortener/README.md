# URL Shortener Images

Please save the uploaded images to this directory with the following names:

1. **First Image (Basic Options)**: Save as `create-short-url-basic.png`
   - This shows the Single Link tab with Long URL, Title, Tags, and Comment fields

2. **Second Image (Advanced Options)**: Save as `create-short-url-advanced.png`
   - This shows the Advanced tab with Expiry Date, Password Protection, Target OS, Target Browser, and Target Geo fields

## How to Add Images

1. Save the first uploaded image as: `create-short-url-basic.png`
2. Save the second uploaded image as: `create-short-url-advanced.png`
3. Place both images in this directory: `/static/img/url-shortener/`

The documentation will automatically display these images once they are saved with the correct filenames.
