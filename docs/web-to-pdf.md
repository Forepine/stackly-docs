## How to Use

1. Navigate to the **Web to PDF** tool in your dashboard.
2. Enter the **URL** of the webpage you want to convert.
3. (Optional) If the page is password-protected, toggle **Authentication** and enter the username/password.
4. Adjust **Page Settings** if needed.
5. Click **Generate PDF**.
6. Once processing is complete, click **Download** to save the file.
